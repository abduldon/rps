# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** web application built using **Flask**, **HTML**, **CSS**, and **JavaScript**. The application allows users to play against the computer with instant results and score tracking.

## 🌐 Live Demo

🔗 https://rps-pk22.onrender.com

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/abduldon/rps.git
```

Move into the project directory

```bash
cd rps
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux/macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
